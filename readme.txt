Prerequisites:
-Machine with Windows
-Node.js
-MongoDB

Instructions:
-Type "npm install" in the terminal
-Type "npm run setup-database"
-Open second terminal and type "npm run test"
-Type "npm run start" in the second terminal
-Nagivate to http://localhost:3000

Notes:
-After running "npm run start" the application may take a long time to start, due to fetching a lot of data to fill the database with initial values
